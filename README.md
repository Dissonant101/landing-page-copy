## Deployment Link
You can access the website here: https://landing-page-copy.vercel.app/. It supports mobile!

## Run Locally
Or you can run it locally using:
```bash
npm i
npm run dev
```
Open http://localhost:3000 with your browser to see the result.

- Next.js 14
- TailwindCSS
- shadcn/ui
